# nodejs-taf-site
陸上競技部用のウェブサイトをNode.jsで作成してます。

## Get Started!
初めて実行するときは、このコマンドを先に実行して下さい。
```
npm run db
```

#### 実環境での実行
```
npm run app
```
`config`フォルダ内に、`product.yml`を作成しておいて下さい。  
作成するときは、`development.yml`を複製してから編集を行って下さい。

#### テスト環境での実行
```
npm run dev
```
`config`フォルダ内に、`development.yml`を作成しておいて下さい（デフォルトはあります）。
